## üö¶ Quick Start Checklist

Before diving into the documentation:

- [ ] Python 3.8+ installed
- [ ] Basic understanding of LLMs and embeddings
- [ ] API keys for LLM provider (OpenAI, Anthropic, etc.)
- [ ] API keys for vector database (if using cloud service)
- [ ] Development environment set up

**Installation:**
```bash
# Core package
pip install llama-index-core

# Essential integrations
pip install llama-index-llms-openai
pip install llama-index-embeddings-openai
pip install llama-index-vector-stores-chroma

# Or use the starter bundle
pip install llama-index
```
